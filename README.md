# Flavour Hive Power BI Dashboard

## Project Overview

This project presents an interactive **Power BI dashboard** for analyzing food delivery data from *Flavour Hive*.
It provides insights into sales performance, user behavior, and city-level trends.

---

## Features

* 1. City-wise sales analysis
* 2. Yearly sales trends
* 3. User gain & loss analysis
* 4. Ratings and order insights
* 5. Category-wise performance (Veg, Non-Veg, Others)

## Dashboards

### 1️. Sales Dashboard

* Total Amount: 987M
* Total Orders: 150K
* Sales trend from 2017–2020
* Top-performing cities

### 2️. City Analysis

* Sales by city
* User distribution by location
* Gain & loss users per city

### 3️. User Analysis

* Active users: 78K
* User demographics (age, gender, marital status)
* Gain vs Lost users breakdown
